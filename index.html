What Is Calcite?

Calcite is Esri's official UI component library — the same LEGO pieces used to build ArcGIS Map Viewer, Field Maps, and Experience Builder. You get pre-built, accessible, mobile-friendly web components with zero CSS from scratch.

WHEN to use Calcite: Standalone HTML apps that should look like Esri products, custom Experience Builder widgets, dashboards with map + filter panels, or any ArcGIS web tool that needs polished UI fast.
Library Use When Example
Calcite Components Standalone HTML apps, custom EB widgets, any ArcGIS-styled UI <calcite-button>
Jimu UI Building Experience Builder widgets (primary choice in EB) <Button> from jimu-ui
ArcGIS Map Components Embedding maps/scenes as web components <arcgis-map>
02

Setup (2 Lines)

No npm. No build tools. Just add these two lines to any HTML file and every Calcite component is available instantly via CDN.

CDN Quick Start Simple
<!-- Add to <head> — that's it --> <script type="module" src="https://js.arcgis.com/calcite-components/2.13.2/calcite.esm.js"> </script> <link rel="stylesheet" href="https://js.arcgis.com/calcite-components/2.13.2/calcite.css"> <!-- Now use any component in <body> --> <calcite-button>Hello Calcite</calcite-button>

Theming

Switch between light and dark mode with a CSS class. Override the brand color with one CSS variable to match any org identity.

WHEN: Matching app to Red Cross branding (swap Esri blue for ARC red), building dark-mode ops dashboards, creating white-label tools for partners.
Light / Dark Mode — one CSS class toggles everything
Simple
Light Mode Dark Mode
Brand Button
This panel respects the current theme.
 A toggle in context
<!-- Light theme (default) --> <div class="calcite-mode-light"> <calcite-button kind="brand">I'm light</calcite-button> </div> <!-- Dark theme — just change the class --> <div class="calcite-mode-dark"> <calcite-button kind="brand">I'm dark</calcite-button> </div> <!-- Or set it on the whole page: --> <body class="calcite-mode-dark">
Custom Brand Color (Red Cross Red) — swap Esri blue for any brand
Simple
Red Cross Action Outline ARC Branded
<style> :root { --calcite-color-brand: #ED1B2E; /* Red Cross red */ --calcite-color-brand-hover: #C41225; /* Darker on hover */ --calcite-color-brand-press: #A00E1E; /* Pressed state */ } </style> <!-- All "kind=brand" components now use Red Cross red --> <calcite-button kind="brand">Red Cross Action</calcite-button>
